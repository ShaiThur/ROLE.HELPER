# ROLE.HELPER

Для запуска сервиса необходимо получить *api* ключ на сайте [Groq](https://groq.com).
Также необходимо создать или получить конфигурационный файл *OpenVPN* формата **.ovpn*, создать папку *proxy* и положить файл в папку.

## Переменные окружения
| Параметр                       | Описание                                                    |
|--------------------------------|-------------------------------------------------------------|
| `SERVICE_NAME`                 | Наименование сервиса                                        |
| `LOG_LEVEL`                    | Уровень логгирования                                        |
| `API_KEY`                      | Ключ для доступа моделей groq                               | 
| `FILE_SIZE_LIMIT_IN_MB`        | Название *LLM*, которая поддерживает *Structured output*    |
| `LLM_STRUCTURED_OUTPUT_NAME`   | Название модели распознавания речи                          |
| `ASR_NAME`                     | Размер голосового файла, который способен обработать сервис |
| `DB_HOST`                      | Хост до базы данных                                         |
| `DB_PORT`                      | Порт базы данных                                            |
| `DB_NAME`                      | Название базы данных                                        |
| `DB_USER`                      | Логин для базы данных                                       |
| `DB_PASSWORD`                  | Пароль для подключения к базе данных                        |
| `PYWAY_TYPE`                   | Тип БД для миграции                                         |
| `PYWAY_DATABASE_MIGRATION_DIR` | Путь до файлов миграции                                     |
| `PYWAY_TABLE`                  | Название таблицы с отслеживанием миграций                   |
| `PYWAY_ENABLED`                | Включение миграции                                          |
